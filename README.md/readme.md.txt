# Food Truck HTML Template

A modern **Food Truck & Street Food HTML Template** designed for restaurants, food trucks, catering services, and street food vendors.

This template includes premium UI sections, responsive layouts, dark mode, and RTL support.

---

# 🚀 Features

- 2 Home Pages
- Fully Responsive Layout
- Dark / Light Mode
- RTL (Right-to-Left) Support
- Premium UI Animations
- Swiper Sliders
- Blog System
- Catering Menu Layout
- Booking Form
- Login / Register Pages
- Premium Documentation

}

### Change Images

---

# 🌙 Dark Mode

The template supports dark mode.

Click the **theme icon in the header** to toggle dark mode.

Dark mode styles are located in:


assets/css/dark-mode.css

---

# 🌍 RTL Support

This template supports **Right-To-Left languages** like Arabic or Hebrew.

Click the **RTL icon in the header** to switch direction.

RTL styles are located in:


assets/css/rtl.css
