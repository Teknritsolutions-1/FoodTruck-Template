Introduction

Thank you for purchasing the Food Truck HTML Template. This template is designed for food trucks, restaurants, and catering businesses.

  • 2 Home Pages
  • Fully Responsive
  • Dark / Light Mode
  • RTL Support
  • Modern UI
  • Swiper Sliders

Installation

  1. Download the template files.
  2. Extract the ZIP archive.
  3. Open index.html in your browser.
